# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.11.x  | Yes       |
| < 1.11  | No        |

## Reporting a Vulnerability

If you discover a security vulnerability, please do **not** open a public GitHub issue.

Instead, report it privately by:
1. Opening a [GitHub Security Advisory](https://github.com/ErlichRonny/chrome-tab-scheduler/security/advisories/new)
2. Or emailing the maintainer directly (see GitHub profile)

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes

You can expect a response within 7 days. If the vulnerability is confirmed, a fix will be prioritized and a new release issued.

## Privacy Note

Tab Scheduler stores all data locally in `chrome.storage.local`. No data is sent to external servers. See [PRIVACY.md](PRIVACY.md) for details.
